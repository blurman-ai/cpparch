## Цель
Убрать FP-other (~17% в iter1): `--diff` репортит переименованный файл как
ADDED-новый-путь ⟵ BASE-старый-путь (тот же файл переехал, не дубль).

## Фикс
`changedSourceFiles`: `git diff --numstat` → `git diff -M --numstat`. Rename
показывается как `old => new` и отсекается существующим кодом → файл не
обрабатывается, фантомная пара не рождается.
