## Context

Right now there is NO duplicate detection in `src/` at all. The whole detector is in the spike:
selective-normalization lexer + parser-free fragmentation (`)`→`{`) + rare-token inverted
index + bag-overlap (gate 0.60) + **mandatory** token-LCS confirm (gate 0.80) +
diff classifier (VERBATIM/RENAMED/EDITED-CONST/EDITED-LOGIC/EXPANDED/SHRUNK).
Plus #071 added clone-density baseline (LD.14) and cross-module matrix (LD.16).

**IMPORTANT:** the spike is actively changing in #071 — start the port ONLY when #071 settles,
otherwise we port a moving target. Before starting: `git log experiments/partial_duplication/`.

## FP guards from #070 (Stage 2) — collapse 602 proposals into ~5 mechanisms

| Mechanism | FP classes it hits | Source |
|-----------|--------------------|--------|
| **idiom stop-list** (logging macros LL_*/ostream chains, Qt scaffolding new Q*Layout/connect, LLView event-handler) | idiom (297) | #070 idiom |
| **AND-gating by contiguous run** (require ≥N consecutive matched tokens, not just bag) | coincidental (100), idiom | #070 coincidental |
| **file-local IDF** (penalty for tokens repeated >K times in a file) | idiom, coincidental | #070 |
| **header-skeleton suppression** (>70% tokens = declarations/access-specifier/Q_* macros → down-weight) | header-impl, whole-file (76) | #070 whole-file/header-impl |
| **data-table / enum-ladder detect** (`case CONST:` / `else if(id==LIT)` ladders → low-entropy) | data-table (45) | #070 data-table |
| **move/rename + generated** (deleted-BASE-path suppress; generated by marker) | other, generated (19) | #070 |

Take fixtures from the **real labeled FP/TP corpus** (`VERIFICATION_REPORT*.md`):
e.g. a confirmed idiom block that fooled the detector → `fixtures/duplication/fail_idiom/`.
