## Методика

### Граф — per-commit, все коммиты с мая 2025
- Инструмент: `experiments/ai_repo_run/generate_per_commit_graph_drift.py` (уже есть).
- Механизм: `archcheck --diff <sha>~1..<sha> .` по **first-parent** истории. Дёшево
  (diff одного коммита, не пере-скан дерева).
- Метрика «ошибки графа»: циклы / cyclic SCC + drift-сигналы (added/removed edges,
  shortcut edges, cycle growth).
- Выход: `.jsonl` (все коммиты) + `.md` (только ненулевой дрифт).

### Копипаст — на HEAD, виновника искать по требованию
- Шаг 1: `archcheck --duplication <repo>` на **HEAD** (после vendor-exclude, коммит e55d0fe).
  Метрика: число reported copy-paste пар + whole-file clone count.
- Шаг 2: **если dup ≠ 0** → для интересных пар искать **коммит-виновник**: `git blame`
  по строкам фрагмента (или bisect), классифицировать тот коммит AI/не-AI.
- Per-commit dup по всей истории **НЕ делаем** (checkout+scan каждого = недели). Только
  целевой drill-down на репах с находками.

### AI-детект — оба сигнала
- Источник списка маркеров: `generate_per_commit_structural_drift.py` → `AI_MARKERS`.
- Сигнал 1 (message/trailer): `co-authored-by: claude|cursor|codex|copilot`,
  `noreply@anthropic.com`, `cursor agent`, `claude code`, `ai-assisted`, `generated by ai`,
  `codex-`, `cursor/`, `copilot`, …
- Сигнал 2 (author): name/email похож на бота (`bot`, `[bot]`, `noreply`).
- AI-коммит = сработал любой сигнал.

## Ключевые решения

### Сессия 2026-06-05
- **Сигнальная иерархия ИИ:** trailer > bot-авторство > AI% по сообщениям ≫ velocity-ramp (мусор —
  ловит любой всплеск, топ-ramp дал AI%=0). **Длина коммит-сообщения** — trailer-независимый прокси,
  ловит невидимый IDE-ИИ и снятые трейлеры.
- **Гиганты с ревью/контрибуторами — выкинуть** (долг от размера, FP); **вендор для dup — обязательно
  фильтровать** (иначе esys/T5ynth ложно «чемпионы копипаста»).
- **2 года истории × размеченный ИИ почти не пересекаются** — агентность феномен 2025; старые репы =
  человеческие или токенные bot-PR. cajeta (2022→2026) — редкое исключение, единственная чистая арка.
- **Демо-тезис не «AI вызвал долг», а «вот агентный C++ без гарда + структурный долг, который тулза
  достаёт»** — причинность не доказуема, специмены доказуемы.

### Исходные
- **Per-commit граф-дрифт: first-parent только** — 280k commits за ~4 часа вместо недель при full history
- **Дупликация на HEAD + blame drill-down** — быстро находим текущее состояние, потом ловим виновника по требованию
- **Bimodal распределение AI%** — не континуум, а два пика: "AI-native" (100%) и "AI-augmented" (50-90%)
- **Коррелирует ли AI% с дрифтом?** — graphillion_kyotodd (99% AI → 101 граф-ошибок) vs fox1245_NeoGraph (94% AI → 114 граф-ошибок) — возможная корреляция, нужна Spearman
